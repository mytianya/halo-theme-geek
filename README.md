## 本地安装
1. 点击链接[下载](https://github.com/mytianya/halo-theme-geek/releases)
2. 进入 Halo 后台 -> 外观 -> 主题
3. 点击右上角 “切换主题”， 在弹出的窗口中选择 “安装主题”
4. 选择下载好的主题包（zip）
5. 安装完成后的主题将会出现在 “未安装” 内，点击启用主题
